# Egogramma

Egogramma è un’opera che registra il tempo come assenza di variazione.

Ogni giorno viene aggiunta automaticamente una nuova data.
Il valore associato è inizialmente zero.

Lo zero non indica neutralità,
ma la normalità del vivere.

## Intervento manuale

L’artista può intervenire **una sola volta al giorno** modificando il valore associato alla data corrente.

Valori ammessi:
- da -3 a +3

Il valore rappresenta una variazione percepita, non misurabile.
Non viene fornita alcuna spiegazione.
Non esiste legenda emotiva.

Se non avviene alcun intervento,
lo zero resta.

## Regola

- nessuna cancellazione
- nessuna riscrittura del passato
- nessun commento
- nessuna interpretazione guidata

Il grafico mostra ciò che c’è stato.
O, più spesso, ciò che non c’è stato.

## Stato dell’opera

L’opera è in corso.
Terminerà quando l’artista smetterà di intervenire.
